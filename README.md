### Introduccion
Pacman es un juego donde se controla un personaje que come capsulas.
El personaje principal debe escapar de 4 fantasmas.
>Los fantasmas son:
 - Fantasma Rojo
 - Fantasma Azul
 - Fantasma Amarillo
 - Fantasma Azul
### Controles

 - Izquierda
 - Derecha
